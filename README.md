## コマンド
・npm i -D ～<br>
　→ ～のインストール

・npm i -D webpack webpack-cli<br>
　→ webpack 本体とコマンドラインインターフェースをインストール

・npx webpack<br>
　→ プロダクションモードでビルド

・npx webpack --mode development<br>
　→ 開発モードでビルド

・npm rum ～<br>
　→ package.json で指定した scripts 内のコマンドを実行

## 注意事項
・プロジェクト名が"webpack"というディレクトリ内に webpack を入れることはできない<br>
